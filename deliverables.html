Deliverable 1

Simple automation

description

Simple workflow that uses the google api to update a google sheet and return all rows.

There is not much to write about this as it is triggered by another workflow, calls the google api to write the name given by the parent workflow and then adds that name to the sheet along with a timestamp. Finally it retrieves all rows from the sheet and returns them to the parent workflow.

challenges

There were some challenges with setting up this workflow as it was my first time using the google cloud console and n8n so there was definetly a learning curve to get started. Once I got familiarized with the google dashboard I didn't have any issues, but the first two to three times I had to setup the credentials I had to rewatch the tutorial.

Deliverable 2

Simple Automation w/LLM

description

This workflow is very very similar to the deliverable shown above with one difference, instead of updating the google sheet with just a name, it now updates the sheet with a name, the origin of that name provided by chatgpt, and a timestamp.

challenges

The learning curve was much gentler for this workflow as n8n is a tool that is designed to work with AI agents calling LLMs so while setting up an openAI account was new, it was a lot simpler than setting up the google cloud console.
